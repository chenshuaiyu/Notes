# 4.对象与类

### 4.1 面向对象程序设计概述

#### 4.1.4 类之间的关系

- 依赖（uses - a）：如果一个类的方法操纵另一个类的对象，就说一个类依赖另一个类。
- 聚合（has - a）：类A的对象包含类B的对象。
- 继承（is - a）：用于表示特殊与一般关系的。

#### 4.3.5 隐式参数和显式参数

```java
class Demo{
    public void show(int a) {
    	System.out.println("a = " + a);
	}
}
```

此方法有两个参数，第一个参数是隐式参数，是出现在方法名前的Demo对象，第二个参数是方法名后面括号中的参数。
